| **Gemini**          | 🟢 Бесплатно | Генерация адаптивных страниц                  |
| **Qwen**            | 🟢 Бесплатно | Быстрая альтернатива, сильный текстовый режим |
| **DeepSeek**        | 🟢 Бесплатно | Производительная open-source модель           |
| **Kimi**            | 🟢 Бесплатно | Хорошо работает с длинными текстами           |
| **Lovable.dev**     | 🟢 Бесплатно | No-code AI-конструктор сайтов                 |
| **Replit**          | 🟢 Бесплатно | Браузерный редактор кода с AI                 |
| **GitHub Pages**    | 🟢 Бесплатно | Бесплатный хостинг .github.io                 |
| **v0.dev (Vercel)** | 🟢 Бесплатно | Генерация React-компонентов                   |

| **Webflow AI**      | 🟠 Платно    | Профессиональный website builder              |

### 📝 Практические сценарии

**A. Хендмейд-продукт — страница презентации + заказ**

- **Инструмент**: Lovable.dev
- **Процесс**: описываете продукт в чате → генерируется страница → авто-деплой
- **Дополнительно**: форма заказа через Formspree (бесплатно)
- **Время**: 10 минут

**B. Ивент / онлайн-курс — лендинг с FAQ**

- **Инструмент**: Claude/ChatGPT
- **Процесс**: описание курса → AI генерирует Hero, Benefits, Program, FAQ, Form
- **Дополнительно**: FAQ создается автоматически из описания
- **Время**: 15 минут

**C. Редактирование кода в браузере**

- **Инструмент**: Replit
- **Процесс**: редактируете код live → «Добавь кнопку WhatsApp» → AI вносит изменения
- **Дополнительно**: бесплатный хостинг на replit.app
- **Время**: 5 минут на изменение

**D. Полный контроль — HTML/CSS/JS**

- **Инструмент**: ChatGPT, Claude, Gemini, Qwen
- **Процесс**: запрашиваете полный код → сохраняете → публикуете на GitHub Pages
- **Дополнительно**: без промежуточных платформ, 100% контроль
- **Время**: 10–15 минут

### 💬 Промпт #1: Генерация лендинга

```text
Создай полную HTML-страницу (single-file) для [ваше описание].

Включи:
1. Семантический HTML со всеми нужными секциями
2. CSS внутри <style> — современный адаптивный дизайн
3. Цвета: [укажите палитру]
4. Минимальный JavaScript для: мобильного меню, плавного скролла, валидации формы
5. Meta-теги для SEO и соцсетей

Код должен быть готов к публикации на GitHub Pages.
Без внешних зависимостей — все inline или через CDN.
```

### 💬 Промпт #2: Лендинг для курса

```text
Создай страницу презентации для моего курса по теме [тема].

Добавь:
- Привлекательный заголовок
- 3 ключевых преимущества
- Программу на [X] недель
- Плейсхолдеры отзывов (3 штуки)
- Форму регистрации: Имя, Email, Телефон
- Современный дизайн, цвета [укажите цвета]
- Видимую CTA-кнопку
```

### 📋 Workflow GitHub Pages

1. Создайте новый репозиторий на GitHub → Public → Create
2. Добавьте файлы → Upload index.html (+ CSS/JS, если отдельно)
3. Включите Pages → Settings → Pages → Branch: main → Save
4. Сайт будет live через 1–2 минуты по адресу: https://username.github.io/repo-name

## 3. Instagram-карусели с Claude AI

### 🎯 Почему Claude AI

- ✅ Генерирует полностью рабочий HTML-код
- ✅ Встроенный экспорт PNG (html2canvas)
- ✅ 100% кастомизация дизайна
- ✅ Без дополнительных затрат
- ✅ Легко переиспользовать шаблон для десятков каруселей

### 🛠️ Необходимые инструменты

| Инструмент                   | Назначение                  | Стоимость                          |
| ---------------------------- | --------------------------- | ---------------------------------- |
| **Claude.ai**                | Генерация HTML-кода         | 🟢 Бесплатно (5–10 каруселей/день) |
| **Браузер (Chrome/Firefox)** | Открытие HTML + экспорт PNG | 🟢 Бесплатно                       |
| **Photopea или Canva**       | Добавление фото на обложку  | 🟢 Бесплатно                       |
| **html2canvas**              | Конвертация HTML → PNG      | 🟢 Бесплатно (встроено)            |

### 💬 Промпт #4: Генератор тем каруселей (RU)

```text
Ты — контент-стратег для Instagram.
Моя ниша: [ниша].
Моя аудитория: [описание].

Придумай 10 тем для каруселей, которые будут активно СОХРАНЯТЬ.

Для каждой темы укажи:
1. Заголовок карусели (цепляющий, с цифрой)
2. Почему это будут сохранять
3. Количество слайдов

Темы должны быть практичными — чек-листы, пошаговые инструкции, подборки, сравнения.
```

### 💬 Промпт #4: Генератор тем каруселей (EN)

```text
You are an Instagram content strategist.
My niche: [niche]
My audience: [description]

Come up with 10 carousel topics that people will actively SAVE.

For each topic provide:
1. Carousel title (catchy, with a number)
2. Why people will save it
3. Number of slides

Topics should be practical — checklists, step-by-step guides, collections, comparisons.
```

### 💬 Промпт #5: Карусель из длинного текста (RU)

```text
Вот мой длинный пост/статья:
[вставить текст]

Преврати его в карусель из 8-10 слайдов.

Правила:
1. Каждый слайд — 1 ключевая мысль
2. Текст на слайде — максимум 30 слов
3. Первый слайд — хук, последний — призыв к действию
4. Добавь визуальные подсказки: где поставить иконку, где выделить цветом, где использовать стрелку
```

### 💬 Промпт #5: Карусель из длинного текста (EN)

```text
Here's my long post/article:
[paste text]

Transform it into a carousel of 8-10 slides.

Rules:
1. Each slide — 1 key idea
2. Text per slide — maximum 30 words
3. First slide — hook, last slide — call to action
4. Add visual hints: where to place icons, where to highlight with color, where to use arrows
```

### 💬 Промпт #6: Полный HTML-дизайн + экспорт PNG (RU)

```text
Создай карусель как HTML-файл размером 1080x1350px на слайд (формат 4:5).

Тема: [тема карусели]
Количество слайдов: [число]
Стиль: [минимализм / яркий / тёмный фон / пастельный]
Цветовая палитра: [цвета или "выбери сам"]
Шрифт: [название или "современный sans-serif"]

Требования к дизайну:
• Каждый слайд: крупный заголовок, 2-3 строки текста, иконка или геометрическая фигура для акцента
• Верстай в одном HTML-файле, где каждый слайд размером 1080x1350px
• Добавь тень к слайдам и обязательно нумерацию в углу
• Первый слайд (обложка) сделай с прозрачным фоном под текстом — я добавлю свою фотку

ВАЖНО:
Добавь JavaScript функцию для экспорта каждого слайда как PNG.
Подключи библиотеку html2canvas.
Создай для каждого слайда кнопку "Сохранить PNG", которая при клике сохраняет именно этот слайд как изображение 1080x1350px.
```

### 💬 Промпт #6: Полный HTML-дизайн + экспорт PNG (EN)

```text
Create a carousel as an HTML file with 1080x1350px per slide (4:5 format).

Topic: [carousel topic]

IMPORTANT:
Add JavaScript function to export each slide as PNG.
Include html2canvas library.
Create a "Save as PNG" button for each slide that saves that specific slide as 1080x1350px image when clicked.
```

- **Дополнительно**: работает на разных языках

- **Инструмент**: NotebookLM

- **Время**: 1–3 минуты

**D. Автоматические субтитры + перевод**

- **Инструмент**: Descript / CapCut AI
- **Процесс**: upload видео → авто-субтитры → экспорт SRT
- **Дополнительно**: 100+ языков, точная синхронизация
- **Время**: 2–5 минут

1. Регистрация → HeyGen.com → Create avatar
2. Upload 2-минутного видео (хороший свет, статичная камера)
3. AI обрабатывает 10–30 минут → аватар готов
4. Создание видео → вставляете скрипт → аватар озвучивает

## 16. Безопасность и защита

### 🛠️ Инструменты

| Инструмент                    | Стоимость          | Описание                              |
| ----------------------------- | ------------------ | ------------------------------------- |
| **Cloudflare**                | 🟢 Бесплатный план | DDoS-защита, HTTPS, кеш               |
| **GitHub Private Repo**       | 🟢 Бесплатно       | Приватный контролируемый код          |
| **Лицензия Creative Commons** | 🟢 Бесплатно       | Базовая защита авторских прав         |
| **.env для API-ключей**       | 🟢 Бесплатно       | Скрытие API-ключей                    |
| **DMCA.com**                  | 🟠 Платно          | Автоматический takedown при воровстве |

### 📊 Риск vs Решение

| Риск                     | Решение                 | Стоимость |
| ------------------------ | ----------------------- | --------- |
| **DDoS-атака**           | Cloudflare (бесплатно)  | Бесплатно |
| **Утечка API-ключа**     | .env + .gitignore       | Бесплатно |
| **Копирование контента** | CC-лицензия + watermark | Бесплатно |
| **Кража кода**           | GitHub Private Repo     | Бесплатно |
| **DMCA takedown**        | DMCA.com уведомления    | Платно    |

### ⚠️ Правило #1: не публикуйте API-ключи

**Никогда** не храните в публичном коде или публичном GitHub:

- OpenAI API keys
- Anthropic API keys
- Google API keys

**Решение**: файлы .env или переменные окружения на сервере

### 💡 Что дает бесплатный Cloudflare

✅ Автоматический HTTPS  
✅ DDoS-защита  
✅ Глобальный кеш  
✅ Блокировка вредоносных ботов

**Настройка**: 10 минут → домен + nameservers → защита активна
