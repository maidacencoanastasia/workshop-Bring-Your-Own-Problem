| **Webflow AI**      | 🟠 Plătit   | Professional website builder       |

### 📝 Scenarii Practice

**A. Produs handmade — pagină de prezentare + comandă**

- **Tool**: Lovable.dev
- **Proces**: Descrii produsul în chat → generare pagină → deploy automat
- **Extra**: Formular comandă cu Formspree (gratuit)
- **Timp**: 10 minute

**B. Eveniment / curs online — landing cu FAQ**

- **Tool**: Claude/ChatGPT
- **Proces**: Descriere curs → AI generează: Hero, Beneficii, Program, FAQ, Formular
- **Extra**: FAQ generat automat din descriere
- **Timp**: 15 minute

**C. Editare cod în browser**

- **Tool**: Replit
- **Proces**: Modifici codul live → "Adaugă buton WhatsApp" → AI modifică
- **Extra**: Hosting inclus gratuit pe replit.app
- **Timp**: 5 minute/modificare

**D. Control total — cod HTML/CSS/JS**

- **Tool**: ChatGPT, Claude, Gemini, Qwen
- **Proces**: Ceri cod complet → salvezi → publici pe GitHub Pages
- **Extra**: Fără platforme intermediare, control 100%
- **Timp**: 10-15 minute

### 💬 Prompt #1: Generare Landing Page

```
Creează o pagină HTML completă (single-file) pentru [descrierea ta].

Include:
1. HTML semantic cu toate secțiunile necesare
2. CSS inline în <style> — design modern, responsive
3. Culori: [specifică paletă]
4. JavaScript minimal pentru: meniu mobil, scroll smooth, validare formular
5. Meta tags pentru SEO și social media

Cod gata de publicare pe GitHub Pages.
Fără dependențe externe — totul inline sau CDN.
```

### 💬 Prompt #2: Landing pentru Curs

```
Creează o pagină de prezentare pentru cursul meu de [temă].

Include:
- Titlu atractiv
- 3 beneficii principale
- Program pe [X] săptămâni
- Testimoniale placeholder (3 buc)
- Formular de înscriere cu: Nume, Email, Telefon
- Design modern, culori [specifică culori]
- Buton CTA vizibil
```

### 📋 Workflow GitHub Pages

1. **Creează repository** pe GitHub → Public → Create
2. **Adaugă fișiere** → Upload index.html (+ CSS/JS dacă separate)
3. **Activează Pages** → Settings → Pages → Branch: main → Save
4. **Site live** în 1-2 minute la: `https://username.github.io/repo-name`

## 3. Carusele Instagram cu Claude AI

### 🎯 De ce Claude AI?

- ✅ Generează cod HTML complet funcțional
- ✅ Export PNG automat integrat (html2canvas)
- ✅ Design personalizabil 100%
- ✅ Zero costuri suplimentare
- ✅ Reutilizabil pentru zeci de carusele

### 🛠️ Instrumente Necesare

| Instrument                   | Scop                         | Cost                          |
| ---------------------------- | ---------------------------- | ----------------------------- |
| **Claude.ai**                | Generare cod HTML            | 🟢 Gratuit (5-10 carusele/zi) |
| **Browser (Chrome/Firefox)** | Deschidere HTML + export PNG | 🟢 Gratuit                    |
| **Photopea sau Canva**       | Adăugare poză pe copertă     | 🟢 Gratuit                    |
| **html2canvas**              | Conversie HTML → PNG         | 🟢 Gratuit (inclus automat)   |

### 💬 Prompt #4: Generator Teme Carusele (RU)

```
Ты — контент-стратег для Instagram.
Моя ниша: [ниша].
Моя аудитория: [описание].

Придумай 10 тем для каруселей, которые будут активно СОХРАНЯТЬ.

Для каждой темы укажи:
1. Заголовок карусели (цепляющий, с цифрой)
2. Почему это будут сохранять
3. Количество слайдов

Темы должны быть практичными — чек-листы, пошаговые инструкции, подборки, сравнения.
```

### 💬 Prompt #4: Generator Teme Carusele (EN)

```
You are an Instagram content strategist.
My niche: [niche]
My audience: [description]

Come up with 10 carousel topics that people will actively SAVE.

For each topic provide:
1. Carousel title (catchy, with a number)
2. Why people will save it
3. Number of slides

Topics should be practical — checklists, step-by-step guides, collections, comparisons.
```

### 💬 Prompt #5: Carusel din Text Lung (RU)

```
Вот мой длинный пост/статья:
[вставить текст]

Преврати его в карусель из 8-10 слайдов.

Правила:
1. Каждый слайд — 1 ключевая мысль
2. Текст на слайде — максимум 30 слов
3. Первый слайд — хук, последний — призыв к действию
4. Добавь визуальные подсказки: где поставить иконку, где выделить цветом, где использовать стрелку
```

### 💬 Prompt #5: Carusel din Text Lung (EN)

```
Here's my long post/article:
[paste text]

Transform it into a carousel of 8-10 slides.

Rules:
1. Each slide — 1 key idea
2. Text per slide — maximum 30 words
3. First slide — hook, last slide — call to action
4. Add visual hints: where to place icons, where to highlight with color, where to use arrows
```

### 💬 Prompt #6: Design Complet HTML + Export PNG (RU)

```
Создай карусель как HTML-файл размером 1080x1350px на слайд (формат 4:5).

Тема: [тема карусели]
Количество слайдов: [число]
Стиль: [минимализм / яркий / тёмный фон / пастельный]
Цветовая палитра: [цвета или "выбери сам"]
Шрифт: [название или "современный sans-serif"]

Требования к дизайну:
• Каждый слайд: крупный заголовок, 2-3 строки текста, иконка или геометрическая фигура для акцента
• Верстай в одном HTML-файле, где каждый слайд размером 1080x1350px
• Добавь тень к слайдам и обязательно нумерацию в углу
• Первый слайд (обложка) сделай с прозрачным фоном под текстом — я добавлю свою фотку

ВАЖНО:
Добавь JavaScript функцию для экспорта каждого слайда как PNG.
Подключи библиотеку html2canvas.
Создай для каждого слайда кнопку "Сохранить PNG", которая при клике сохраняет именно этот слайд как изображение 1080x1350px.
```

### 💬 Prompt #6: Design Complet HTML + Export PNG (EN)

```
Create a carousel as an HTML file with 1080x1350px per slide (4:5 format).

Topic: [carousel topic]
Number of slides: [number]
Style: [minimalist / bright / dark background / pastel]
Color palette: [colors or "choose yourself"]
Font: [name or "modern sans-serif"]

Design requirements:
• Each slide: large title, 2-3 lines of text, icon or geometric shape for accent
• Code in one HTML file where each slide is 1080x1350px
• Add shadow to slides and mandatory numbering in corner
• First slide (cover) make with transparent background under text — I'll add my photo

IMPORTANT:
Add JavaScript function to export each slide as PNG.
Include html2canvas library.
Create a "Save as PNG" button for each slide that saves that specific slide as 1080x1350px image when clicked.
```

### 💬 Prompt #7: Serie de Carusele în Stil Uniform (RU)

```
Создай 5 каруселей на темы:
1. [тема 1]
2. [тема 2]
3. [тема 3]
4. [тема 4]
5. [тема 5]

Сделай всё как ОДИН HTML-файл.
Все карусели в едином визуальном стиле:

Единые элементы:
• Один шрифт (выбери современный sans-serif)
• Одна цветовая палитра (предложи профессиональную)
• Одинаковое расположение нумерации слайдов
• Одинаковый стиль заголовков
• Одинаковый размер текста

Обложки:
Первый слайд каждой карусели делай с областью под фото — я потом подставлю свою фотку в Canva за 2 минуты.

Экспорт:
Добавь кнопки для сохранения каждого слайда как PNG 1080x1350px. Используй html2canvas.
```

### 💬 Prompt #7: Serie de Carusele în Stil Uniform (EN)

```
Create 5 carousels on topics:
1. [topic 1]
2. [topic 2]
3. [topic 3]
4. [topic 4]
5. [topic 5]

Make everything as ONE HTML file.
All carousels in unified visual style:

Unified elements:
• One font (choose modern sans-serif)
• One color palette (suggest professional)
• Same slide numbering position
• Same title style
• Same text size

Covers:
```

### 📋 Workflow Complet: De la Zero la Postare

1. **Scrii Prompt 6 în Claude** → obții HTML (2 min)
2. **Salvezi ca "carusel.html"** → deschizi în browser (1 min)
3. **Apeși "Salvează PNG"** pentru fiecare slide (2 min)
4. **Deschizi slide 1 în Canva** → adaugi poza ta (3 min)
5. **Upload pe Instagram** ca carusel (2 min)

⏱️ **Timp total: 10-15 minute**

## 16. Securitate și Protecție

### 🛠️ Instrumente

| Instrument                   | Cost            | Descriere                    |
| ---------------------------- | --------------- | ---------------------------- |
| **Cloudflare**               | 🟢 Plan Gratuit | Protecție DDoS, HTTPS, cache |
| **GitHub Private Repo**      | 🟢 Gratuit      | Cod privat, controlat        |
| **Licență Creative Commons** | 🟢 Gratuit      | Protecție copyright simplă   |
| **.env pentru chei API**     | 🟢 Gratuit      | Ascunde chei API             |
| **DMCA.com**                 | 🟠 Plătit       | Takedown automat pentru furt |

### 📊 Risc vs Soluție

| Risc                 | Soluție                  | Cost    |
| -------------------- | ------------------------ | ------- |
| **Atac DDoS**        | Cloudflare gratuit       | Gratuit |
| **Cheie API expusă** | Fișier .env + .gitignore | Gratuit |
| **Conținut copiat**  | Licență CC + watermark   | Gratuit |
| **Cod furat**        | GitHub Private Repo      | Gratuit |
| **DMCA takedown**    | DMCA.com notificări      | Plătit  |

### ⚠️ Regula #1: Nu Expune Chei API

**Niciodată** în cod public sau GitHub public:

- OpenAI API keys
- Anthropic API keys
- Google API keys

**Soluție**: Fișiere `.env` sau variabile de mediu pe server

### 💡 Cloudflare Gratuit Include:

✅ HTTPS automat
✅ Protecție DDoS
✅ Cache global
✅ Blocare boți malițioși

**Setup**: 10 minute → domeniu + nameservers → protecție activă
